> **Q1 Qt 的信号槽使用方法是怎么样的？**

<br>

在 Qt5 中，`QObject::connect()`有五个重载：

<br>

``` cpp
QMetaObject::Connection connect(const QObject *,
                                const char *,
                                const QObject *,
                                const char *,
                                Qt::ConnectionType);

QMetaObject::Connection connect(const QObject *,
                                const QMetaMethod &,
                                const QObject *,
                                const QMetaMethod &,
                                Qt::ConnectionType);

QMetaObject::Connection connect(const QObject *,
                                const char *,
                                const char *,
                                Qt::ConnectionType) const;

QMetaObject::Connection connect(const QObject *,
                                PointerToMemberFunction,
                                const QObject *,
                                PointerToMemberFunction,
                                Qt::ConnectionType)

QMetaObject::Connection connect(const QObject *,
                                PointerToMemberFunction,
                                Functor);
```
